Project Overview: Architected a unified data platform by integrating Microsoft Dynamics 365 CRM data with Azure for advanced analytics.

Cloud Stack: Azure Synapse, Data Factory (ADF), Dynamics 365.

Engineering Highlights:

ETL Automation: Built data pipelines using Azure Data Factory to synchronize transactional data from CRM to a centralized data warehouse.

Data Unified Platform: Enabled massive scale analytics by connecting data lineage directly to Azure Synapse Analytics.

Security: Implemented granular access controls to ensure secure data movement across enterprise environments.
